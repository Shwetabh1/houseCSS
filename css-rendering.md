Every browser uses a css rendering engine to draw the HTML/CSS web page.

For Safari it is : Webkit.
Chrome -> Blink a fork of Webkit.
FireFox -> Gecko

IE
Engine: Trident
CSS-prefix: -ms
Edge
Engine: EdgeHTML
CSS-prefix: -ms
Firefox
Engine: Gecko
CSS-prefix: -moz
Opera
Engine: Presto → Blink1
CSS-prefix: -o (Presto) and -webkit (Blink)
Safari
Engine: WebKit
CSS-prefix: -webkit
Chrome
Engine: WebKit → Blink2
CSS-prefix: -webkit
